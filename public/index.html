Users - routes

By log in you can add, edit and remove your own toys , Admin has the ability to delete and modify users and their toys.

Stydy request - routes

Some "GET" requests allow you to retrieve information about study requests. You can use the following queries to customize your results:

  • PerPage: Specify the number of study requests per page.
  • Page: Navigate through different pages of study requests.
  • Sort: Define the sorting criteria (e.g., "_id", "topics").
  • Reverse: Choose the sorting direction, yes is to do reverse.

We verifies that min study duration is grater than 5 the maximum study time is greater than the minimum and also that the duration of the study is greater than or equal to the time that the person defined as hours he can study.

#Method URL Body Description
GET /studyRequests/requestsList none Get a list of all the study, will present 10 results per page.
GET /studyRequests/myStudyRequests none get all study requests that you posted - by token
GET /studyRequests/single/:id none Get the user by id
GET /studyRequests/search?topic=bible&language=english none search for a study request by searching follwing name after the "topic=" or "language=", it will present 10 results per page.
GET /studyRequests/duration?min=12&max=25 none Get a list of the study request by min & max hourse , it will present 10 results per page.
GET /studyRequests/topic/:topicName none Get a list of all study requests according to the selected topic category, it will present 10 results per page.
